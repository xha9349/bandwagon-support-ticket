## 搬瓦工 Support Ticket 是什么？用来干什么的？

**搬瓦工 support ticket（工单）**：用户在后台发起的一对一书面求助请求，客服审阅后通过系统和邮件回复，是搬瓦工与用户之间的主要官方沟通渠道。

搬瓦工是自我管理型（self-managed）VPS，不提供全托管服务。所以客服能解决的问题是有范围的，主要包括：

- 账户层面的问题：付款异常、账号被锁、续费问题
- 硬件/网络层面的故障：节点宕机、数据中心级故障
- 退款申请（30天退款政策范围内）
- IP 更换申请（付费）
- 特殊需求：购买额外 IP、套餐升级、机房迁移问题

以下问题提了也没用，客服不会介入：Linux 系统配置、软件安装、建站报错——这些属于你自己的服务器，自己解决。

## 一、提交搬瓦工工单的完整步骤

### 前提：确认能登录账号

没有账号或者登不进去，就只能发邮件：**support@bandwagonhost.com**。但优先推荐工单系统，处理效率更有保障。

### 第一步：进入工单系统

登录搬瓦工后台（右上角 Client Area），点击顶部菜单的 **Support → Tickets**。

也可以直接访问：`bandwagonhost.com/supporttickets.php`

进去之后能看到所有历史工单的列表。

### 第二步：新建工单

点击页面上的 **Open New Ticket** 按钮，进入填写页面。

### 第三步：填写工单内容

工单页面需要填写以下字段：

1. **Subject**（主题）：用一句话概括你的问题，例如 "Payment successful but VPS not activated"
2. **Related Service**（关联服务）：选择对应的 VPS 主机；没有相关 VPS 则选 None
3. **Message**（内容）：详细描述问题，越具体越好，包括：出现问题的时间、错误提示截图说明、你已经尝试过的排查步骤
4. **Attachments**（附件）：可以上传截图、日志文件、PDF 等——能说明问题的截图往往能大幅缩短处理时间

### 第四步：提交并等待

点击 **Submit** 提交工单，然后就是等。客服回复后，你绑定的邮箱会收到提醒。

## 二、工单必须用英文写吗？

必须。搬瓦工没有中文客服，工单系统只受理英文内容。

不会英文没关系——用百度翻译或谷歌翻译，先把问题用中文写清楚，再翻译成英文粘贴进去就行。翻译质量哪怕一般，客服也能看懂你的意思。核心是把问题说清楚，别让客服猜。

> **一个实用的工单模板：**
>
> Subject: [简短问题描述]
>
> Hello,
>
> I am writing to report an issue with my VPS. [描述问题发生的时间和具体现象]
>
> My VPS ID is: [在 KiwiVM 后台可以找到]
>
> I have already tried: [你已做过的排查]
>
> Please help me resolve this issue. Thank you.

## 三、搬瓦工工单多久回复？几个关键信息

这个问题没有固定答案。回复速度取决于三个变量：提交时间、问题复杂度、当前工单总量。

从实际用户反馈来看，工作日提交的简单问题，快则 1 小时内就能收到回复，一般在 24 小时内。

有一点必须注意：**搬瓦工客服周六、周日不上班**。工作时区是加拿大温哥华时间（比北京时间晚 16 小时）。周末提交的工单，需要等到周一才会被处理。

算一下：如果你北京时间周五晚上 10 点提交工单，温哥华那边是周五早上 6 点，还算在工作日。但如果周六才提，就要等到他们周一上班，也就是北京时间周二凌晨才开始处理。

## 四、提交工单后最容易踩的坑

**不要反复回复同一工单。** 这是最多人犯的错误。搬瓦工的工单系统按照"最新回复时间"排序——你每次回复，工单就会重新排到队伍末尾。结果就是越催越慢，工单永远在往后推。

提交了就放手等，直到客服回复为止。

---

**遇到"Martin C"回复怎么办？**

这是第二个坑。搬瓦工有一个客服名叫 Martin C，社区里不少用户怀疑这是一个自动化机器人——每次出现，回复都是答非所问的模板内容。

如果你的工单收到了 Martin C 的回复，看看回复内容是否真的解决了你的问题。没有的话，把你的原始问题完整复制一遍，重新回复一次，下一次通常会有真人客服接手。

## FAQ：关于搬瓦工工单的常见问题

**Q：没有搬瓦工账号，能联系客服吗？**

A：可以，直接发邮件到 support@bandwagonhost.com 即可。但注意要说清楚你的具体问题和相关账号信息（注册邮箱）。

**Q：搬瓦工工单可以用中文写吗？**

A：不建议。工单系统面向英文客服，用中文写大概率会被忽略或处理时间更长。用翻译工具转成英文，成本很低，效果更好。

**Q：工单提交后一直没有回复，怎么办？**

A：先确认是否在工作日提交（周末不处理）、以及是否反复回复导致排队靠后了。等满 48 个工作小时后仍无回复，可以礼貌地补充一次追问。

**Q：搬瓦工客服能帮我配置服务器软件吗？**

A：不能。搬瓦工是无管理型 VPS，服务器内部的软件配置、系统问题需要自行解决。客服只处理账号、付款、硬件/网络层面的问题。

**Q：提交退款申请也是通过工单吗？**

A：是的。30 天退款政策内，在后台提交退款申请（通过工单或专用退款页面），一般处理后退回原支付渠道。
